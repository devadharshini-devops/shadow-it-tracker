# Shadow IT Tracker - AI Based Detection System

## Overview
An AI-powered system that identifies unauthorized applications within organizations and provides real-time risk analysis to prevent data leakage.

## Live Demo
https://shadow-it-tracker.vercel.app

## Features
- **Real-time App Verification**: Flags unregistered apps like Instagram, Dropbox instantly
- **AI Risk Scoring**: Classifies threats as High/Medium/Low risk with percentage
- **Activity Logs**: Tracks all verification attempts for IT audit
- **Royal Business Dashboard**: Clean UI for IT administrators

## Tech Stack
Frontend: HTML, CSS, JavaScript, Poppins Font
Backend: Node.js, Express.js
Design: Royal Business Dashboard theme
